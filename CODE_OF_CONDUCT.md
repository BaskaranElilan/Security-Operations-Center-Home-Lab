# Code of Conduct

## Our Commitment

This project is committed to providing a welcoming and inclusive environment for all contributors and participants, regardless of background or identity. We are dedicated to providing a harassment-free experience for everyone.

## Expected Behavior

- **Be Respectful**: Treat all participants with dignity and respect
- **Be Inclusive**: Welcome diverse perspectives and backgrounds
- **Be Constructive**: Provide helpful, actionable feedback
- **Be Professional**: Maintain professional language and tone
- **Be Accountable**: Take responsibility for your actions

## Unacceptable Behavior

The following behaviors are not tolerated:
- Harassment, discrimination, or bullying of any kind
- Offensive comments related to gender, age, race, religion, or sexuality
- Threats or violence of any kind
- Posting or threatening to post private information without consent
- Spamming or trolling
- Deliberately disrupting discussions
- Inappropriate sexual content or advances
- Any other conduct that violates laws or commonly accepted standards

## Reporting & Enforcement

If you witness or experience unacceptable behavior:

1. **Document**: Note what happened, when, and who was involved
2. **Report**: Contact the project maintainer:
   - Email: baskaran.elilan@example.com
   - GitHub: Use private message if available
3. **Confidentiality**: Reports will be handled confidentially
4. **Resolution**: Appropriate action will be taken

Possible actions:
- Warning to participant
- Temporary ban from participation
- Permanent ban from project
- Reporting to GitHub trust and safety team

## Scope

This Code of Conduct applies to:
- GitHub issues and pull requests
- Discussions and comments
- Project meetings and forums
- Any official project communications
